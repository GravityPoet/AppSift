# AppSift app icon

The blue-and-white eraser with colored fragments is AppSift's canonical app-icon brand.

- Canonical master: `AppSift/Assets.xcassets/AppIcon.appiconset/icon_512@2x.png`
- Website/favicon copy: `docs/assets/icon.png`
- Source treatment: centered 1080 × 1080 crop from the approved 1254 × 1254 artwork, offset 87 px from each edge, then resized to 1024 × 1024.
- Invariants: preserve the eraser, fragments, colors, lighting, proportions, and dust details. Remove only the external white canvas; don't redraw or substitute the artwork.
- Every smaller macOS icon is derived from the canonical 1024 × 1024 master.
